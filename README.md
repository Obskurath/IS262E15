## ▶️ Cómo abrir y ejecutar

### 1) Abrir la solución
1. Abre Visual Studio.
2. Ve a **File → Open → Project/Solution**.
3. Selecciona: [`IS262E15Ejercicio06.sln`](IS262E15Ejercicio06.sln)

### 2) Compilar
- En Visual Studio selecciona **Build → Build Solution** ✅

### 3) Ejecutar un proyecto
1. En el *Solution Explorer*, elige el proyecto que quieras correr.
2. Clic derecho → **Set as Startup Project** ⭐
3. Presiona **F5** (con depuración) o **Ctrl + F5** (sin depuración) ▶️
